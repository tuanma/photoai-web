QR CODE
TYPES, USES & EXAMPLES

There are no types of QR codes. A QR code contains a character string as payload. The QR code is always just a QR code, i.e. a coded representation of a chain of bytes.
TEXT: Hello! this is plain text example.
LOCATION: geo:<lat>,<long>,<alt>
example: geo:40.71872,-73.98905,100
vCard:
BEGIN:VCARD
VERSION:2.1
N:Franz Testhuber
FN:Testhuber;Franz
TITLE:Professor
TEL;CELL:+1 555 1234 56
EMAIL:HOME;INTERNET:ich@du.da
URL:https://exmaple.com/zeuq.html
ADR:;irgendwo;weg;Berlin;;12345;Germany
END:VCARD
SMS: SMSTO:<nummer>:<text>
example: SMSTO:4915112345678:Hurga
EVENTS: BEGIN:VEVENT
SUMMARY:Biergarten
DTSTART:20190105T220000Z
DTEND:20190105T110000Z
END:VEVENT
E-MAIL: MATMSG:TO:ich@du.da;
SUB:Betreff;
BODY:Hallo da draussen;;
WIFI ACCESS: WIFI:S:tenda;T:WPA;P:dsh@sd;;
Network Name  Network Type  WiFi Password
CALL: tel:<nummer>
